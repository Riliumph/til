# どんなログがほしい

- システムログ
- リクエストログ
- アプリケーションログ
- 統計情報

## システムログ

AWS などのインフラのログ。  
たとえな、AWS のオートスケーリングが動かないなど。

## リクエストログ

不正発見やフォレンジックのログ。  
不正なアクセスであるか、正しい処理を行えているかといったトレーサビリティのためのログ。

- 不正発見の場合は、異常系ログが大事
- フォレンジックの場合は、正常系ログが大事

## アプリケーションログ

アプリが死んだときや正しく動作しなかったときの情報を発信するログ。

## 統計情報

Zabbix や AWS CloudWatchLogs などで算出する資源の使用率などのログ。
