# 君やってたよね

こうなるSQLを紹介する。

```sql
CREATE TABLE tests (
  test_id bigint GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
  test_name text unique
);

INSERT INTO tests (test_name) VALUES ('hoge');
INSERT INTO tests (test_name) VALUES ('fuga');
```

このテーブルに対して、`select`を発行したい。  
でも、偶然にもスペースキーが壊れてしまった！！どうしよう！？  

こんな状況で以下のSQLを作り始めたら、さすがに「君、やってたよね？」となってしまう。

```sql
SELECT/**/*/**/FROM/**/tests;
```

この部分コメントを使用した構文はSQLでは以下のように解釈される。

```sql
SELECT
*
FROM
tests;
```
